# My Custom Keybinds

A tiny personal Chrome extension for browser and site keybinds.

## Keybinds

Keybinds live in `keybinds.js`. The help overlay is rendered from that
registry, so new bindings show up there automatically.

Current bindings:

| Keybind | Action |
| --- | --- |
| `Alt+C` | Clone the current tab |
| `Alt+K` | Show the keybind reference |

## Install

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Choose "Load unpacked".
4. Select this folder.

Chrome may ask you to confirm or reassign `Alt+C` or `Alt+K` at
`chrome://extensions/shortcuts` if another shortcut already owns it.
